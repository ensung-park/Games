# Games

A collection of browser-based games.

## Sweet Match

A polished match-3 puzzle game with 30 levels, special candies, and cloud save.

**Play:** [https://ensung-park.github.io/Games/sweet-match/](https://ensung-park.github.io/Games/sweet-match/)

### Features
- 30 levels across 3 difficulty tiers
- Special candies: Striped, Wrapped, Color Bomb
- Special combo effects (cross clear, 5x5 explosion, etc.)
- Blockers: Ice, Chocolate, Stone
- Google account sign-in with cloud save
- Leaderboard (top 20 players)
- Guest mode with local save
- Smooth animations (swap, gravity, dissolve, bounce)
- Procedural sound effects (Web Audio API)
- Works on desktop and mobile (touch + mouse)

### Tech
- Single HTML file, no build tools
- HTML5 Canvas rendering at 60fps
- Firebase Auth + Cloud Firestore
- GitHub Pages hosting
